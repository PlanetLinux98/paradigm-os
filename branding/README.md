# branding/

Visual identity source files:

- `wallpapers/` — the rotating light/dark wallpaper set.
- `icons/` — logo, favicon, and any custom icon-theme overrides.
- `theme/` — GNOME Shell / GTK theme (including the official high-contrast
  variant) and Plymouth boot-splash theme.

Every asset here must be checked against WCAG contrast guidelines before being
finalized — see the accessibility commitments in `docs/SPEC.md`.
